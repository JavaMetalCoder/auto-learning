# Esercizio: Cambia Colore del Titolo

## Descrizione
In questo esercizio, creeremo una semplice pagina web che contiene:
- Un titolo (`<h1>`).
- Un pulsante (`<button>`).

Quando l'utente clicca sul pulsante, il colore del titolo cambierà in modo casuale.

## Obiettivi
- Utilizzare HTML per creare la struttura della pagina.
- Applicare stili di base con CSS.
- Implementare la logica per il cambio di colore con JavaScript.

## Struttura dei File
- `index.html`: Contiene la struttura della pagina.
- `style.css`: Contiene gli stili della pagina.
- `script.js`: Contiene il codice JavaScript per il cambio di colore.

## Istruzioni
1. Crea un file `index.html` e aggiungi un titolo e un pulsante.
2. Collega il file CSS per gli stili.
3. Collega il file JavaScript per la logica.
4. Implementa una funzione JavaScript che genera un colore casuale e lo applica al titolo quando il pulsante viene cliccato.

## Suggerimenti
- Usa `Math.random()` per generare colori casuali.
- Modifica la proprietà `style.color` del titolo per cambiare il colore.
